## 零、geojson.io 设计系统速查（节选自 latest.md）

### 色彩
```
品牌蓝: #007afc (light) / #3195ff (dark)
中性灰: gray-50...#900 (Tailwind scale)
Light 页底: #fff  Dark 页底: #262626
Light 边框: #e5e5e5  Dark 边框: #404040
```

### 圆角
```
2px (超小/工具按钮) → 4px (默认/卡片) → 6px (弹窗) → 8px (模态框)
```

### 阴影 (Light only, Dark = none)
```
sm: 0 1px 2px 0 rgb(0 0 0/.05)
md: 0 4px 6px -1px rgb(0 0 0/.1)
lg: 0 10px 15px -3px rgb(0 0 0/.1)
顶部工具栏: 0 2px 10px 2px rgba(0,0,0,0.1)
```

### 字体
```
正文: Open Sans / Inter, 400/600/700, 16px, antialiased
代码: Source Code Pro, 14px
小字: 12px (辅助说明)
```

### 按钮层级
```
Primary:   蓝底白字, hover 深蓝
Secondary: 透明底灰字, hover 浅灰底
Icon:      透明底, hover 浅灰底
Danger:    红底红字
```

### 过渡
```
150ms cubic-bezier(.4,0,.2,1) — 所有交互
400ms ease-out — 弹窗动画
```

### 空间分配
```
地图 ≥ 60% 视口 — 始终优先
编辑面板 ~320px 固定宽度
工具栏 64px 固定高度
```

```
┌──────────────────────────────────────────────────────────┐
│  情绪地图 v1.0    [导入] [导出]  │  [范围] [图层] [底图] [⚙] │
└──────────────────────────────────────────────────────────┘
```

| 属性 | 值 |

| background | `#ffffff` |
| border-bottom | `1px solid #e5e5e5` |
| shadow | `0 2px 10px 2px rgba(0,0,0,0.1)` |

## 四、弹窗瘦身计划

参照 geojson.io 弹窗设计模式：

| 当前弹窗 | 新方式 | 原因 |
|----------|--------|------|
| [DATA] 数据源 | 工具栏 [导入] → 弹窗（Export 模式） | 文件选择需空间 |
| [OV] 数据概览 | 左面板"数据"段展开详情 | 信息展示，无需弹窗 |
| [TB] 数据表格 | 保留弹窗 | 表格需大空间 |
| [RNG] 分析范围 | 工具栏 [范围] → 面板型弹窗（Layers 模式） | 从按钮附近浮出 |
| [GV] 数据治理 | 左面板"分析"段内操作 | 管道操作，面板内完成 |
| [Map] 底图切换 | 工具栏 [底图] → 面板型弹窗（Layers 模式） | 底图缩略图网格 |
| [LY] 图层控制 | 左面板"图层"段 | 实时可见，无需弹窗 |
| [ANA] 情绪分析 | 左面板"分析"段 → 点击展开配置表单 | 面板内完成 |
| [*] 设置 | 工具栏 [⚙] → 小型弹窗 | 低频操作 |

**弹窗分类**：
- **模态弹窗**（有遮罩）：导入、导出、数据表格、设置
- **面板型弹窗**（无遮罩）：范围、底图
- **面板内嵌**（零弹窗）：数据概要、图层控制、分析工具、数据治理
