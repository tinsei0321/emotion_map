# 视觉中转站 (Vision Inbox)

本目录用于**跨 Chat 图片识别中转**。

## 用法

1. 在另一个支持图片的 Claude Chat 中，发送图片并让它描述内容
2. 让那个 Chat 将描述结果写入本目录下的文件（编号递增）
3. 本项目的 Claude 会自动读取最新文件

## 文件约定

| 文件 | 用途 |
|------|------|
| `latest.md` | 最新一次识图结果（会被覆盖） |
| `001.md` ~ `NNN.md` | 历史识图记录（不覆盖） |
| `inbox.md` | 待处理队列（追加模式） |

## 另一个 Chat 的写入指令

告诉另一个 Chat：

> 把图片内容描述写入文件，路径是 `docs/vision-inbox/latest.md`
> 格式用：
> ```
> # 识图结果 — [时间戳]
> [图片描述]
> ```

## 本项目 Claude 的读取方式

我会自动读取 `docs/vision-inbox/latest.md`。也可以告诉我要读哪个编号。
