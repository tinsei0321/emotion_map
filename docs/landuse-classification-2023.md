## 分类体系（三级编码）

| 层级 | 代码位数 | 示例 | 数量 |
|---|---|---|---|
| 一级类 | 2 位 | `07` 居住用地 | 24 |
| 二级类 | 4 位（前 2 位=一级） | `0701` 城镇住宅用地 | 111 |
| 三级类 | 6 位（前 4 位=二级） | `070101` 一类城镇住宅用地 | 40 |

- 24 一级类覆盖：耕地/园地/林地/草地/湿地/农业设施/居住/公管公服/商服/工矿/仓储/交通/公用设施/绿地开敞/特殊/留白/陆地水域/渔业用海/工矿通信用海/交通运输用海/游憩用海/特殊用海/其他土地/其他海域。
- **三级类仅 06-12 城镇建设相关一级类设立**（40 个）；耕地/园地/林地/草地/湿地/陆地水域/各用海类等仅设到二级。
- 16 留白用地、24 其他海域 无二级类。

> **关于"140 三级类"**：指南 P5 述"113 二级类、140 三级类"，但本 PDF 表 2.1 + 附录 A 实际提取 111 二级 / 40 三级。差异疑为表格合并单元格提取遗漏或指南统计含其他附录（B 城镇村工矿 / E 地下空间补充分类）。**以本 PDF 实际内容为准**；如需补全 113/140，重读 PDF 附录 A 逐行校勘。

## 使用（Python API）

```python
from ai_qa.landuse_codes_2023 import (
    landuse_name, landuse_level, landuse_parent, landuse_children, landuse_search,
    LANDUSE_L1, LANDUSE_L2, LANDUSE_L3, EMC_PRESET_TO_GB,
)

landuse_name('07')        # '居住用地'
landuse_name('0701')      # '城镇住宅用地'
landuse_name('070101')    # '一类城镇住宅用地'
landuse_level('070101')   # 3
landuse_parent('070101')  # '0701'
landuse_children('07')    # ['0701','0702','0703','0704']
landuse_search('商业')    # [('09','商业服务业用地',1),('0901','商业用地',2),...]
```

## 与 EMC 用地预设对照

EMC 现有用地预设（`DATA/boundaries/presets/manifest.json` 的 `land_*`，按 `DLMC` 字段落色）是本国标子集。对照见 `EMC_PRESET_TO_GB`：

| EMC 预设 DLMC 值 | 国标一级 | 国标二级 |
|---|---|---|
| 商业 | 09 商业服务业用地 | 0901 商业用地 |
| 居住 | 07 居住用地 | 0701 城镇住宅用地 |
| 公园广场 | 14 绿地与开敞空间用地 | 1401 公园绿地 |

## 与字段语义层的关系
规划中的字段语义层（`land_use_class` role）以本文件代码+名称为值域——上传用地数据的 DLMC/地类名称字段识别为 `land_use_class` role 后，其值可经本表对到国标代码，供 4×5 归因、用地落色、跨源统一引用。
